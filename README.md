# TankTourneyClient

python client for [Tank Tourney](https://github.com/adarshmelethil/TankTourney).

## Install client 

`python setup.py install`

`pip install -e .`

## Example client

First arguement is player number, 1 or 2

`python example/Random.py 1`

`python example/Random.py 2`
